# SLD样式

GeoServer 中主要使用 SLD样式（Styled Layer Descriptor：风格化图层描述器）。

特点：

- XML 格式的样式文件
- 扩展名为 `.sld`



它还有一些扩展的样式属性，如 CSS Styleing、YSLD Styling、MB Styling。
