# Layer Group

图层组（Layer Group）适用于减小图层请求量，用一个简单的请求获取多层复杂数据。

Layer Group Mode：

|          Mode          |    译名    | 是否可以命名 | 是否包含子项 | 是否列出子项 | 备注         |
| :--------------------: | :--------: | :----------: | :----------: | :----------: | ------------ |
|         Single         |   单图层   |      是      |      否      |      否      |              |
|    Opaque Container    | 不透明容器 |      是      |      是      |      否      | 隐藏子项     |
|       Name Tree        |   命名树   |      是      |      是      |      是      |              |
|     Container Tree     |   容器树   |      否      |      是      |      是      |              |
| Earth Observation Tree | 地球观察树 |      是      |      是      |      是      | 要求有根图层 |
