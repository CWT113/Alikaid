::: danger 数组的缺点

- **数组一旦创建，其长度就固定了**。如果要存储更多的元素，就必须重新创建一个更大的数组，并将原有元素复制过去，效率不高；
- **数组只能存储同一类型的数据**，如果要存储不同类型的数据，只能使用 Object 类型，但这样失去了类型安全性，容易出现类型转换错误；
- 数组提供的方法较少，**对于添加、删除、插入等操作非常不便**，且效率不高；
- 数组的 length 属性只能告诉我们数组的容量，而不能直接获取实际存储的元素个数；

:::
